Sprint 26 – KICK-1657 Prevent GOEs applying using a gateway's Kickstart Scheme ID

  • The route that went into development
Notes and previous versions for UR and links to Mural boards
  • There was an issue whereby employers who had previously applied under a gateway organisation were applying for additional jobs (called a variation) but using their gateway's Kickstart ID in the application process, instead of either going through their gateway or, submitting a new application themselves. This work discourages them from going down this route. Note that we don't have any validation on the service so we can't wholly prevent this from happening.
  • The start page has been updated following a UR session. The alternative version can be found here. This was the original route 1 for A/B testing.
